Q 1.var counter = 0; counter++; ++counter; alert( counter ); What will alert this and why?

Answer
Ans: alert( counter ) will display 2. Initially counter is 0 then after counter++ it will increase by 1 and after applying prefix increment operator its value will be 2.

Q 3. What are the final values of all variables a, b, c and d after the code below? var a = 1, b = 1; var c = ++a; var d = b++;

Answer
Ans: a=2(prefix operator first increses value of a to 1 and then returns 2)
c=2(a becomes 2 after applying prefix increment operator and then its value assigned to c)
d=1(postfix increment operator returns the original value first and then increments var by 1 so b assigns its original value to d and then it becomes 2 )

Q 6. Rewrite this if using the ternary operator '?': if (a + b < 4) { result = 'Below'; } else { result = 'Over'; }

Answer
Ans: (a + b < 4) ? 'Below' : 'Over';

Q 7. Rewrite following to switch var a = +prompt('a?', ''); if (a == 0) { alert( 0 ); } if (a == 1) { alert( 1 ); } if (a == 2 || a == 3) { alert( '2,3' );

Answer
Ans: var a = +prompt('a?', '');
switch(a)
{
case 0:
alert( 0 );
break;
case 1:
alert( 1 );
break;
case (a == 2 || a == 3):
alert( '2,3' );
break;
default:
alert("nothing");
break;
}
